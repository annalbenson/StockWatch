# StockWatch

<b>Description</b>
- At-a-glance stock tracking app using data from Market Watch (https://www.marketwatch.com/)
- 

<b>Installation</b>
- USB Cable
- Android device or emulator with minimum SDK 21
- Android Studio or equivalent IDE


<b>Usage</b>

    </p>
